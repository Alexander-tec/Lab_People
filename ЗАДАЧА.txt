Задача со списком граждан

- в файле находится список с информацией о людях
- ФИО, Дата рождения, Дата смерти (если есть), пол.
- ID - внутренний уникальный идентификатор записи (число).
- Указаны связи родства -- отец, мать. По ID. (Какой-то информации может не быть).
- Прочая информация

Получить информацию из файла (разного формата) и совершить некоторую обработку данных.

Варианты задач по обработке данных:

На циклы:
а) Для конкретного человека (по ID) указать текущий возраст (*?)
б) Найти человека в списке по части ФИО (*?)
в) Найти всех людей с указанным днем рождения
г) Найти самого старого человека
д) Посчитать сирот (не указаны отец и мать)
е) Найти сколько людей родились до 1941

ж) Вычислить в каком месяце больше всего ДР
з) Узнать процент ныне живущих из всех указанных в списке
и) Вычислить у скольки (в %) человек день рождения в (указанном пользователем) месяце
к) Узнать количество братьев и сестер у человека (по ID)
л) Найти человека с наибольшим количеством детей
м) Проверить, есть ли у человека дядя или тётя

Обходы по фамильному дереву:
+ Найти самого дальнего предка от ID 

Варианты ввода данных:
- В формате csv таблицы (в текстовом файле построчно значения, разделенные символом ";" )
- В формате JSON
- В таблице на веб-странице
+ В онлайн-таблице google sheets

В случае наличия аргументов программа получает их:
- через аргументы командной строки
- через консоль (интерактивно)
- через propmt
- через поле в web-странице

Варианты вывода данных:
- вывести на экран
- вывести в соседний файл
- вывести на веб-страницу
+ дополнить файл информацией
